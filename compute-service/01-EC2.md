# Elastic Compute Cloud (EC2)
If you are interested in tech, maybe you have heard about EC2 but mostly what you heard is EC2 instance. This two kinda different because EC2 instance is one of many features in EC2. It provides on-demand, **scalable computing capacity** in AWS. With EC2 we can launch as many or as few virtual server as we want, configure network and security based on what we need, and determine the storage required.

You will hear the word scalable a lot in this repo because this is what makes cloud provider superior than the other. You can add capacity (scale up) to handle more compute-heavy task, such as spikes in website traffic, costumer increase on our system, or even data-processing that will take time to finish. You can also reduce the capacity (scale down) again when the usege decrease.
